# getHardwareStack


## Description
Hardware configuration information list

## Request method
GET

## Request address
https://idata-jmr-api.jcloud.com/v1/regions/{regionId}/hardwareStack

|Name|Type|Required or not|Default value|Description|
|---|---|---|---|---|
|**regionId**|String|True||Region ID|
