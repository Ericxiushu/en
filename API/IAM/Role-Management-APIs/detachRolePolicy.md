# detachRolePolicy


## 描述
Associate policy for pole

## 请求方式
POST

## 请求地址
https://iam.jdcloud-api.com/v1/role/{roleName}/policy:detachRolePolicy

|名称|类型|是否必需|默认值|描述|
|---|---|---|---|---|
|**roleName**|String|True| |Role Name|

## 请求参数
|名称|类型|是否必需|默认值|描述|
|---|---|---|---|---|
|**policyJrn**|String|False| |Policy jrn|


## 返回参数
|名称|类型|描述|
|---|---|---|
|**requestId**|String| |


## 返回码
|返回码|描述|
|---|---|
|**200**|OK|
